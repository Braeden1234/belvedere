Format Reference
ATC_Map Syntax
The ATC_Map index tab defines your project structure. Projects, branches, folders, and files are declared using a simple tag syntax that RenderDoc-X parses at load time.
// Tab 0 — ATC_Map index $Project(My App) %Branch(main) %Folder(src) %FileType(html) #INDEX { .index(@t.1) } %FileType(css) { .style(@t.2) } %FileType(js) { .app(@t.3) } %FileType(scss) { .theme(@t.4) }

Use Cases
What Can You Build?
RenderDoc-X turns any Google Doc into a live web app host. Use Hot Doc Connect for real-time dev workflows, or Doc Connect for static deploys — no server required.
// Hot Doc Connect — live reload mode const poll = async () => { const changed = await checkHashes(fileMap); if (changed) pushUpdate(); }; setInterval(poll, 1000); // Doc Connect — static mode const html = buildFromContents( contents[htmlFilename], contents ); applyHTML(html, filename); // no setInterval — loads once
