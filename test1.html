ATC_Map Syntax
The ATC_Map index tab defines your project structure. Projects, branches, folders, and files are declared using a simple tag syntax that RenderDoc-X parses at load time.
// Tab 0 — ATC_Map index $Project(My App) %Branch(main) %Folder(src) %FileType(html) #INDEX { .index(@t.1) }
